My Projects

Automatic Fish Feeder

This project involves designing an automatic fish feeder using a **servo motor** to dispense food at scheduled intervals. The system uses a microcontroller (like an Arduino) to control the servo motor, which rotates and releases food into the fish tank. The feeder can be programmed to operate at specific times of the day to ensure that fish are fed on time, even when the owner is not around.

A Seismograph

This project uses sensors to create a **seismograph** that can detect vibrations or tremors in the ground. It consists of a sensor module and a display system that allows you to track and visualize the vibrations in real-time. This system can be used to monitor seismic activity and warn about potential earthquakes or disturbances in a specific area.
